# Solar-System-project

#### Description:
This is a project, developed during the CS50 course. It is a mini game-project for simulation of our Solar system.
You suppose to say which planets you want to simulate and then setup the asteroid/comets to throw inside the system.
Then - just enjoy the graphical simulation result.

#### Requirements:
Before running the project.py, make sure you install required libraries. Required libraries are indicated in Requirements.txt file.
